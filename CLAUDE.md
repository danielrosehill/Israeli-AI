The purpose of this repo is to gather together a list (hopefully I will update at least occasionally) of MCP servers that might be of interest to those living in Israel.

Method:

I will provide lists of projects from Github and elsewhere as well as some notes. 

You can help me by organising the information into a useful README.md.

When doing so, adhere to my general preferences for creating resource lists:

- Organse by section 
- Simple styling: I like to list Github repos with shields.io badges, copy short desriptions, add a start count and last commit 
- If I have notes about projects, I will add them. These can appear after the descriptions in italics to make clear that they are my own notes and opposed to those by the author 
- Some MCPs will have websites and demos/POCs. Organise into separate sections

## Template

Here's the template I like to adhere to:

# Section

Description of this section. Like: MCP servers that might be useful to those living in Israel.

## Repo 1 

Shields.io badge, Star Count, Last Commit (Live)

Description
